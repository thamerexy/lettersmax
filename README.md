# Letters Max

Letters Max is a premium Arabic trivia game where two teams (**Red vs. Green**) compete to connect sides of a honeycomb board by answering questions correctly.

## Features

- **Animated Tutorial**: Integrated guide explaining core rules.
- **Branding**: High-resolution logo with seamless blending.
- **Stay Awake**: Built-in feature to prevent screen dimming during gameplay.
- **Real-time Sync**: Powered by Supabase for smooth multiplayer experience.

## Tech Stack

- **Frontend**: Vite, React, TypeScript, Tailwind-like custom CSS.
- **State Management**: Zustand.
- **Backend/Real-time**: Supabase.

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```
