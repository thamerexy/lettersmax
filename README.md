# Letters Max - Version 1.0.0

Letters Max is a strategic multiplayer Arabic trivia game designed for high-performance and real-time interaction using a GoDaddy PHP/MySQL backend.

## Features
- **Real-time Gameplay**: Synchronization between Admin and Players via optimized polling.
- **Arabic Trivia**: Support for thousands of questions across multiple categories.
- **PWA Ready**: Installable on mobile and desktop devices with offline support.
- **Admin Dashboard**: Comprehensive control over game rounds, team assignments, and question management.

## Technical Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: Vanilla CSS with modern glassmorphism design.
- **Backend**: PHP 8.x (GoDaddy Shared Hosting compatible)
- **Database**: MySQL

## Deployment
Automated deployment is configured via GitHub Actions to GoDaddy FTP.

### Configuration
Update `public/api/config.php` with your MySQL credentials to connect the backend.
